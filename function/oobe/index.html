欢迎来到简·搜索！这是一个轻量级的搜索入口，支持搜索建议、天气显示和快捷链接管理。

  • 选择你喜欢的配色方案，让界面更舒适。
  • 开启或关闭天气显示，控制页面信息呈现。
  • 设置搜索建议点击行为，获得更符合习惯的体验。
  • 稍后可在设置页面继续调整更多选项。

隐私政策

我们不会收集你的搜索数据。你的搜索内容仅会与搜索建议提供商 Microsoft Bing 共享，因此数据仅用于搜索建议服务。

如果你启用天气显示，位置信息会与 OpenWeatherMap 共享，用于获取本地天气信息。

所有设置和数据均存储在本地，目前没有在线同步功能。
